Centos7安装apisix

2021年4月28日
| linux
| 阅读
文章目录
    • 1.rpm安装
    • 2.修改配置文件
    • 修改dashboard配置文件
    • 启动服务

1.rpm安装

yum install yum-utils
yum-config-manager --add-repo https://openresty.org/package/centos/openresty.repo
yum install -y openresty curl git gcc openresty-openssl111-devel unzip  #如果服务器安装了kong这步会有冲突，需要先卸载kong
rpm -e kong-community-edition-1.1.0-1.noarch
然后继续安装
yum install -y https://github.com/apache/apisix/releases/download/2.5/apisix-2.5-0.x86_64.rpm
安装dashboard
yum install https://github.com/apache/apisix-dashboard/releases/download/v2.6/apisix-dashboard-2.6-0.x86_64.rpm

2.修改配置文件

cd /usr/local/apisix/conf/
config-default.yaml
conf.yaml
tips:etcd是集群的话就填集群的client地址，如：
etcd:
  host:                           # it's possible to define multiple etcd hosts addresses of the same etcd cluster.
    - "http://172.16.100.23:2379"     # multiple etcd address, if your etcd cluster enables TLS, please use https scheme,
    - "http://172.16.100.30:2379"     # multiple etcd address, if your etcd cluster enables TLS, please use https scheme,
    - "http://172.16.100.31:2379"

修改dashboard配置文件

cd /usr/local/apisix/dashboard/conf
conf:
  listen:                 # yamllint disable rule:comments-indentation
     host: 0.0.0.0  # the address on which the `Manager API` should listen.
                          # The default value is 0.0.0.0, if want to specify, please enable it.
                          # This value accepts IPv4, IPv6, and hostname.
    port: 9000            # The port on which the `Manager API` should listen.
  #allow_list:             # If we don't set any IP list, then any IP access is allowed by default.
  #  - 127.0.0.1           # The rules are checked in sequence until the first match is found.
  #  - ::1
  
  
    etcd:
    endpoints:            # supports defining multiple etcd host addresses for an etcd cluster
      - 172.16.100.23:2379
      - 172.16.100.30:2379
      - 172.16.100.31:2379
